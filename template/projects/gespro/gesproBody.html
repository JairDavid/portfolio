Gespro - (Gestor de Proyectos)

Lets see the roles into the application and what they can do.

ROLE

Project manager

  • - Can manage people assigned to projects
  • - Can view progress of each project
  • - Can view project deliverables

ROLE

Human resources

  • - Can manage user credentials and their roles.
  • - Can register new users into application.

ROLE

Controller

  • - Can view projects.
  • - Can add phases to projects.
  • - Can assign a corresponding percentage on each phase and assign its deliverables.

ROLE

Coordinator

  • - Can manage projects.
  • - Can estimate costs and calculate a deliverable time
